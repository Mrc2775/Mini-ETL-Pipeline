# Mini ETL Pipeline 

This is a data engineering project using Python, SQLite, SQL, and the Open Library API.

## What this project does

- Fetches book data from the Open Library API
- Cleans and stores the data in a SQLite database
- Runs SQL queries on the data
- Exports query results to CSV files

## Technologies Used

- Python
- requests
- sqlite3
- SQL
- CSV

## Files

- `mini_etl.py` - main Python script
- `books.db` - SQLite database file
- `results_by_year.csv` - books count by year
- `results_top_authors.csv` - top 5 authors by book count

## Project Goal
Practicing Extract, Transform, Load (ETL) thinking with Python and SQL.
